# Try multiple URL patterns for downloading

Try multiple URL patterns for downloading

## Usage

``` r
try_download_patterns(base_url, patterns, end_year, file_type)
```

## Arguments

- base_url:

  Base URL for downloads

- patterns:

  Vector of filename patterns to try

- end_year:

  School year end

- file_type:

  Type of file (for messaging)

## Value

Data frame if successful, NULL otherwise
