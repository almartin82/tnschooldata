# Get assessment cache file path for given year and type

Get assessment cache file path for given year and type

## Usage

``` r
get_assessment_cache_path(end_year, type, level = "all")
```

## Arguments

- end_year:

  School year end

- type:

  Data type ("tidy" or "wide")

- level:

  Optional level filter that was used ("all", "state", "district",
  "school")

## Value

Full path to cache file
