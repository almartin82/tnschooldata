# Read assessment data from cache

Read assessment data from cache

## Usage

``` r
read_assessment_cache(end_year, type, level = "all")
```

## Arguments

- end_year:

  School year end

- type:

  Data type ("tidy" or "wide")

- level:

  Optional level filter that was used ("all", "state", "district",
  "school")

## Value

Cached data frame
